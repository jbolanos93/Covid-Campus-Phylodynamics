## Repository Structure
- **`notebooks/`** — Jupyter notebooks reproducing analyses and figures  
- **`scripts/`** — Python, Bash, and R scripts  
- **`data/`** — Example dataset (BEAST MCC tree; 20 trees for rapid processing)
